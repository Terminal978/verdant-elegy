# 🎨 Шпаргалка по эмодзи для команды

## Иерархия ролей (от высшей к низшей)

1. 👑 **Создатель / Владелец** - высшая роль
2. 🏆 **Главный администратор** - руководит администрацией
3. 🛠️ **Модератор** - следит за порядком
4. ⚙️ **Администратор** - помогает с управлением

## Советы

1. Используйте одинаковые эмодзи для одной роли
2. Не перегружайте - один эмодзи на карточку
3. Выбирайте понятные и узнаваемые символы
4. Сохраняйте единый стиль во всей команде
