# 📖 Руководство по редактору команды

## Как редактировать состав команды

1. Войдите в админ-панель (`admin.html`)
2. Нажмите кнопку **"👥 Редактировать команду"**
3. Используйте панель инструментов для добавления элементов
4. Нажмите **"👁️ Предпросмотр"** для проверки
5. Нажмите **"💾 Сохранить"** для применения изменений

## Панель инструментов

### Форматирование текста
- **B** - Жирный текст (выделите текст и нажмите)
- **I** - Курсив (выделите текст и нажмите)
- **U** - Подчеркнутый (выделите текст и нажмите)

### Структурные элементы
- **H3** - Добавить заголовок категории
- **👤** - Добавить члена команды (с диалогом)
- **📁** - Добавить новую категорию
- **📝** - Добавить вступительный блок
- **⚡** - Добавить блок призыва к действию

### Дополнительно
- **—** - Горизонтальная линия-разделитель
- **↵** - Перенос строки

## Быстрое добавление члена команды

При нажатии на кнопку **👤** появятся диалоги:
1. Введите имя члена команды
2. Введите роль/должность
3. Введите описание обязанностей
4. Выберите эмодзи для аватара (👑 ⚙️ 🛠️ 🎮 ⚡)

Код автоматически вставится в нужном формате!

## Доступные CSS классы

### Структура страницы

- `.team-intro` - Вступительный блок с описанием команды
- `.team-category` - Категория членов команды (Создатели, Админы, Модераторы)
- `.team-grid` - Сетка для карточек членов команды
- `.team-join` - Блок призыва к действию (присоединиться к команде)

### Карточка члена команды

- `.team-member` - Контейнер карточки
- `.member-avatar` - Аватар (эмодзи или изображение)
- `.member-info` - Информация о члене команды
- `.member-name` - Имя члена команды
- `.member-role` - Роль (должность)
- `.member-description` - Описание обязанностей

## Пример структуры

```html
<div class="team-category">
    <h3>👑 Создатель проекта</h3>
    <div class="team-grid">
        <div class="team-member">
            <div class="member-avatar">👑</div>
            <div class="member-info">
                <h4 class="member-name">Имя</h4>
                <p class="member-role">Должность</p>
                <p class="member-description">Описание обязанностей</p>
            </div>
        </div>
    </div>
</div>
```

## Советы

- Используйте эмодзи для аватаров: 👑 ⚙️ 🛠️ 🎮 ⚡
- Группируйте членов команды по ролям
- Добавляйте краткие, но информативные описания
- Сохраняйте единый стиль оформления
